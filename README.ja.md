## 🤔 なぜAntiKitなのか？

あなたは最強のAIコーディングエージェント **Google Antigravity** を持っています。しかし、時々こう感じませんか？
- 😥 **専門的すぎる:** 「コンテキスト」「スキーマ」「プロンプト」… 頭が痛い！
- ⚠️ **英語ばかり:** 日本語で聞いても、英語（または不自然な日本語）で返ってくる。
- 😨 **リスクが高い:** もし `C:\Windows` フォルダを削除されたら？
- 😵 **カオス:** 今誰と話してるの？コーダー？テスター？

**AntiKit** はこれらすべてを解決します。Antigravityの生のエンジンを、誰でも運転できる **高級スポーツカー** に変えます。

---

## ✨ 「Vibe Coding」の4つの柱

### 1. 🌏 ネイティブ・フルエンシー (あなたの言葉で話す)
不自然な翻訳はもう終わり。AntiKitはAIに、100% **日本語で思考し、応答** するように強制します。
> *以前: "Here is your code" -> 今: "コードが出来上がりました、ボス！"*

### 2. 🛡️ ゼロ・フィアー (安心サンドボックス)
安心してコードを書けます。**デジタル・フォースフィールド** を設置しました:
- 🚫 **破壊的コマンドのブロック:** `rm -rf` や `format` は禁止です。
- 🔒 **プロジェクト・ロックダウン:** AIはプロジェクトフォルダの外には手出しできません。
- 🧱 **システム保護:** Windowsやシステムのファイルは聖域です。

### 3. 🆔 クラリティ・バッジ (チームを知る)
AntiKitはすべてのAIエージェントに名札を付けます。誰が働いているか常に分かります。
> `> 🆔 Agent: @architect (アーキテクト) | 🛠️ Skills: 計画, 設計`

### 4. 🧠 スーパーバイザー・ブレイン (自己修正)
AIに「良心」を植え付けました。コードを出す前に、AIは密かに自問します:
> *「待てよ、このコードでいいのか？シニアエンジニアは承認するかな？」*
あなたが見る **前に**、AIは自分のミスを修正します。

## 🚀 1クリックでインストール

テックウィザードである必要はありません。ターミナルを開いて、この **魔法の呪文** を貼り付けるだけです:

### 🪟 Windows (PowerShell)
```powershell
iwr https://raw.githubusercontent.com/hasugoii/antikit/main/install.ps1 -useb | iex
```

### 🍎 macOS / 🐧 Linux
```bash
curl -fsSL https://raw.githubusercontent.com/hasugoii/antikit/main/install.sh | bash
```

*(インストール中に、言語を選択してください: **Japanese (日本語)**)*

## 🎮 どうやって "Vibe" する？

インストールしたら、チャットでこれらの `/slash` コマンドを入力して専門家を召喚しましょう:

| コマンド | 召喚... | こんな気分の時に... |
| :--- | :--- | :--- |
| `/brainstorm` | 💡 **アイデアマン** | 「アイデアはあるけど、どこから始めればいいか...」 |
| `/plan` | 📝 **アーキテクト** | 「コードを書く前にしっかりした計画が欲しい。」 |
| `/visualize` | 🎨 **デザイナー** | 「見た目を美しくしたい。」 |
| `/code` | 💻 **シニアDev** | 「コードを書いてくれ。面倒くさい。」 |
| `/deploy` | 🚀 **DevOps** | 「これをインターネットに公開して！」 |
| `/audit` | 🏥 **ドクター** | 「私のコードは健康？安全？」 |
| `/recap` | 🧠 **秘書** | 「今まで何してたっけ？思い出させて。」 |

## 🛠️ プロフェッショナルのためにも ⚡

AntiKitは初心者向けだと思っていますか？ **考え直してください。** これはシニア開発者のための強化外骨格です：

1.  **⚡ 爆速の効率化:** 面倒なセットアップ、設定、ボイラープレートを1秒で自動化します。
2.  **🧠 40以上の専門スキル:** エージェントには以下の深い知識がプリロードされています：
    *   **技術:** Next.js, React, Node.js, Python, Docker, Cloudflare.
    *   **方法論:** TDD, Clean Code, DDD, セキュリティパターン.
    *   **専門分野:** SEO, UX/UI, パフォーマンス分析.
3.  **👮 規律の強制:** 嫌われ役になることなく、チーム全体で **Linting、Testing、Safety** のルールを強制できます。
4.  **🧘 ディープ・フロー:** AIがバックグラウンドで（危険な）作業を安全に処理している間、あなたはゾーンに入り続けることができます。
