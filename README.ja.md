## 🎯 AntiKit とは？

**AntiKit**（Antigravity Kit）は [Google Antigravity](https://ai.google.dev/aistudio) のための究極の **Vibe Coding** ツールです。

> AI開発のための個人用OS - ワークフロー、エージェント、スキルがあなたのアイデアを本番対応コードに変換します。

**次のようなAIエンジニアに最適：**
- 🚀 AI駆動のワークフローで高速デリバリー
- 🧠 セッション間でコンテキストを失わない
- 🛡️ 初日から本番対応コードを作成

## 😫 問題

**AIでコーディングする際、こんな問題に直面していませんか？**

<table>
<tr>
<td width="50%">

### 🧠 「AIが全部忘れた！」

20-30メッセージ後、AIは技術スタックやデータベーススキーマを忘れます。コードを捏造し、重複テーブルを作成し始めます。

</td>
<td width="50%">

### 💥 「とりあえず動く」

AIは「とりあえず動く」コードを提供。エラー処理なし、シークレットのハードコード、エッジケース無視。本番デプロイ = 💥

</td>
</tr>
<tr>
<td>

### 🔄 「Aを直したらBが壊れた」

終わりのないデバッグループ。AIは根本原因を見つけずに盲目的にパッチ。コードは乱れ、最終的に削除して最初からやり直し。

</td>
<td>

### ❓ 「次に何をすべき？」

次のステップがわからず画面を見つめる。ロードマップなし、明確なプロセスなし。ただランダムにチャット。

</td>
</tr>
<tr>
<td>

### 🍝 「スパゲッティコード」

AIは整理せずにコードを追加し続ける。数千行のファイル、入れ子の関数。1週間後、自分のコードが理解できない。

</td>
<td>

### 😰 「変更が怖い、バックアップなし」

変更したいが他を壊すのが怖い。Gitを知らない、バックアップなし。1つのミス = すべての作業が無駄に。

</td>
</tr>
</table>

## ✅ 解決策

**AntiKitは規律あるワークフローでこれらすべての問題を解決します。**

| 問題 | AntiKitの解決策 |
|------|-----------------|
| 🧠 AIの記憶喪失 | `/save-brain` でコンテキストを `.brain/` に保存 |
| 💥 脆弱なコード | `/code` にセキュリティチェックとベストプラクティス |
| 🔄 デバッグループ | `/debug` で根本原因を特定 |
| ❓ 迷子状態 | `/next` で最適な次のアクションを提案 |
| 🍝 乱雑なコード | `/refactor` + `/audit` でコードを整理 |
| 😰 バックアップなし | `/rollback` + 自動Git統合 |

## 📋 ワークフローパイプライン

AntiKitは開発を**4つの明確なフェーズ**に整理します：

### フェーズ1: 🎯 計画
> **まだコードを書かないで！**

| コマンド | 目的 |
|----------|------|
| `/init` | プロジェクト構造、Git、環境のセットアップ |
| `/recap` | 休憩後のコンテキスト復元 |
| `/brainstorm` | 計画前のアイデア探索 |
| `/plan` | データベース、API設計、仕様作成 |
| `/next` | 行き詰まった時の提案 |

### フェーズ2: 🔨 構築
> **安全で美しいコード**

| コマンド | 目的 |
|----------|------|
| `/visualize` | 美しいUIレイアウトの設計 |
| `/code` | セキュリティチェック付きコーディング |
| `/run` | アプリケーションの実行 |

### フェーズ3: ⚙️ 運用
> **保守と最適化**

| コマンド | 目的 |
|----------|------|
| `/debug` | バグの根本原因を特定 |
| `/test` | テストの実行 |
| `/refactor` | コードの整理 |
| `/rollback` | 以前のバージョンに戻す |
| `/deploy` | 本番環境へデプロイ |

### フェーズ4: 🛡️ 管理
> **管理と改善**

| コマンド | 目的 |
|----------|------|
| `/save-brain` | プロジェクト知識を保存 |
| `/config` | skillsとagentsの設定 |
| `/audit` | セキュリティとコードレビュー |
| `/ak-update` | AntiKitの更新 |

アイデアから本番まで完全な開発ライフサイクル。

専門家：`@frontend`、`@backend`、`@database`、`@security`、`@devops`、`@tester`、`@debugger`、`@performance`、`@architect`、`@doc`、`@orchestrator`、`@pentester`、`@mobile`、`@game`、`@seo`、`@explorer`

コンテキストに応じて自動ロードされる深い知識モジュール。

## 💡 プロのヒント

| タイミング | アクション |
|------------|----------|
| 🌅 毎朝 | `/recap` でコンテキスト復元 |
| 🌙 セッション終了時 | `/save-brain` で知識を保存 |
| ✅ 機能完了 | `/save-brain` + Gitコミット |
| 🆕 新規プロジェクト | `/init` → `/recap` |
| 😕 行き詰まった？ | `/next` で提案を取得 |
| 🔧 大きな変更前 | 先にGitコミット！ |
