## ⚡ ソリューション

**AntiKitは欠けていたピースです。**

**20のワークフロー**、**16のAIエージェント**、**40の専門スキル**で構成されるフレームワークが、Antigravityを強力なツールから完全な開発オペレーティングシステムに変えます。

| AntiKitなし | AntiKitあり |
|-------------|-------------|
| 「AIがまたスキーマを忘れた」 | `/recap`が完全なコンテキストを復元 |
| 修正のたびにランダムなバグ | `/code`に自動テストを含む |
| 「次に何をすれば？」 | `/next`が最適なパスを提案 |
| コードレビュー？それ何？ | `/audit`がセキュリティと品質をチェック |
| 何も変更するのが怖い | `/rollback`で任意のバージョンに戻る |
| セッション間でコンテキストが失われる | `/save-brain`が知識を永続化 |

## 🚀 クイックスタート

**⏱️ 30秒でインストール：**

### macOS / Linux

```bash
curl -fsSL https://raw.githubusercontent.com/hasugoii/antikit/main/install.sh | bash
```

### Windows (PowerShell)

```powershell
irm https://raw.githubusercontent.com/hasugoii/antikit/main/install.ps1 | iex
```

**インストール後、Antigravityを再起動して試してください：**

```
/recap
```

## 📋 コマンド

AntiKitはAI駆動開発を明確なジャーニーに整理します：

### 🎯 フェーズ1：計画
> *コードを書く前に考える*

| コマンド | 機能 |
|----------|------|
| `/init` | プロジェクト構造、Git、環境をセットアップ |
| `/recap` | 休憩後にコンテキストを復元 — AIがすべてを覚えている |
| `/brainstorm` | コミット前にAIとアイデアを探る |
| `/plan` | データベース、APIを設計、詳細なスペックを作成 |
| `/next` | 行き詰まった？インテリジェントな提案を取得 |

### 🔨 フェーズ2：構築
> *初日から本番対応コードを書く*

| コマンド | 機能 |
|----------|------|
| `/visualize` | AIで美しいUIレイアウトをデザイン |
| `/code` | ビルトインセキュリティチェック付きでコードを書く |
| `/run` | アプリケーションを起動 |

### ⚙️ フェーズ3：運用
> *品質を維持し、問題を迅速に修正*

| コマンド | 機能 |
|----------|------|
| `/debug` | 症状だけでなく根本原因を見つける |
| `/test` | 包括的なテストを実行 |
| `/refactor` | 乱雑なコードを自動的にクリーンアップ |
| `/rollback` | 以前のバージョンに安全に戻る |
| `/deploy` | 自信を持って本番環境にプッシュ |

### 🛡️ フェーズ4：管理
> *プロジェクトを健全に保つ*

| コマンド | 機能 |
|----------|------|
| `/save-brain` | プロジェクト知識を保存 — AIは決して忘れない |
| `/config` | スキル、エージェント、言語設定を構成 |
| `/audit` | セキュリティレビューとコード品質チェック |
| `/ak-update` | AntiKitを最新バージョンに更新 |
| `/uninstall` | 必要に応じてクリーンに削除 |

## 🤖 含まれるもの

### 20のワークフロー
アイデアから本番までの完全な開発ライフサイクル。各ワークフローがステップバイステップでガイド。
