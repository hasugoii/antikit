# 🚀 AntiKit 工作流套件 v1.0

**Vibe Coders 的完整工作流系统** - 只需有想法，AI处理一切。

> 💡 **理念：** 你负责创意和决策。AI处理所有技术细节 - 包括你不知道需要的。

---

## 📋 命令列表 (20个工作流)

### 🌟 启动 & 上下文
| 命令 | 描述 | 处理的盲点 |
|------|------|-----------|
| `/init` | 创建完整的新项目 | 环境变量, Git, 代码质量工具 |
| `/recap` | 恢复上下文 | 上下文恢复 |
| `/save-brain` | 保存知识 | API文档, 变更日志 |
| `/config` | 配置技能和代理 | 资源优化 |

### 🎯 功能开发
| 命令 | 描述 | 处理的盲点 |
|------|------|-----------|
| `/brainstorm` | 规划前探索想法 | 市场研究, MVP |
| `/plan` | 设计全面功能 | 认证, 数据库, 图表 |
| `/visualize` | 设计UI/UX | 加载/错误状态, 无障碍 |
| `/code` | 编写高质量代码 | 安全, 验证 |

### ⚙️ 运维
| 命令 | 描述 |
|------|------|
| `/run` | 启动应用 |
| `/test` | 检查逻辑 |
| `/deploy` | 部署到生产环境 |

### 🔧 维护
| 命令 | 描述 |
|------|------|
| `/debug` | 修复bug |
| `/refactor` | 清理代码 |
| `/audit` | 健康检查 |
| `/rollback` | 回滚到之前版本 |

---

## 🎮 推荐工作流

### 📦 新项目
```
/init → /plan → /visualize → /code → /run → /test → /deploy → /save-brain
```

### 🌅 每天开始
```
/recap → /code → /run → /test → /save-brain
```

---

## 💡 Vibe Coder提示

1. **自然说话** - AI会询问缺失的信息
2. **不怕犯错** - 有 `/rollback`
3. **每天结束: `/save-brain`** - 明天不会丢失上下文
4. **定期 `/audit`** - 预防胜于治疗
