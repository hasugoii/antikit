# 工作流: /ak-update - AntiKit更新 (Enhanced)

> **Context:** Agent `@devops`
> **Required Skills:** `server-management`, `git-workflow`

AntiKit更新管理器。检查并更新，支持包选择功能。

---

## 阶段 1: 获取注册表

检查版本并获取注册表。

---

## 阶段 2: 与本地比较

读取 `~/.gemini/antikit_installed.json` 并比较。

---

## 阶段 3: 显示可用更新

```
📦 ANTIKIT 更新中心

┌─────────────────────────────────────────────────────┐
│ 📂 工作流 ({count} 更新)                             │
├─────────────────────────────────────────────────────┤
│ [ ] 1. /debug         1.0.0 → 1.2.0  ⭐ 热门更新    │
│ [ ] 2. /code          1.1.0 → 1.2.1  🆕 新版本      │
└─────────────────────────────────────────────────────┘

📊 您的语言: 🇨🇳 中文
```

---

## 阶段 4: 选择

```
🔽 选择要更新的内容:

输入数字 (逗号分隔) 或:
• all - 全部更新
• workflows - 仅工作流
• cancel - 取消

> _
```

---

## 阶段 5-6: 执行和确认

下载选定的包并安装。

---

```
1️⃣ 浏览更多包? /ak-browse
2️⃣ 查看更新历史? /ak-history
3️⃣ 贡献包? /ak-contribute
```
