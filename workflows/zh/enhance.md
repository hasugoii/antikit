# /enhance - 更新应用

$ARGUMENTS

---

## 任务

此命令为现有应用添加功能或进行更新。

### 步骤:

1. **了解当前状态**
   - 使用 `python .agent/scripts/session_manager.py info` 加载项目状态
   - 了解现有功能和技术栈

2. **规划变更**
   - 确定要添加/更改的内容
   - 检测受影响的文件
   - 检查依赖关系

3. **向用户展示计划**（重大变更时）
   ```
   "要添加管理面板:
   - 将创建15个新文件
   - 更新8个文件
   - 大约需要10分钟

   可以开始吗？"
   ```

4. **执行**
   - 调用相关代理
   - 实施变更
   - 测试

5. **更新预览**
   - 热重载或重启

---

## 使用示例

```
/enhance 添加暗色模式
/enhance 构建管理面板
/enhance 集成支付系统
/enhance 添加搜索功能
/enhance 编辑个人资料页面
/enhance 做响应式适配
```

---

## 注意事项

- 重大变更需获得批准
- 对冲突请求发出警告（如：项目使用PostgreSQL时要求"使用Firebase"）
- 每次变更用git提交
