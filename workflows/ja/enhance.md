# /enhance - アプリケーション更新

$ARGUMENTS

---

## タスク

このコマンドは既存アプリケーションに機能を追加または更新します。

### ステップ:

1. **現状把握**
   - `python .agent/scripts/session_manager.py info` でプロジェクト状態を読み込む
   - 既存機能、技術スタックを理解する

2. **変更計画**
   - 何を追加/変更するか決定
   - 影響を受けるファイルを検出
   - 依存関係を確認

3. **ユーザーに計画を提示**（大きな変更の場合）
   ```
   「管理パネルを追加するには:
   - 15個の新規ファイルを作成
   - 8個のファイルを更新
   - 約10分かかります

   始めてよろしいですか？」
   ```

4. **適用**
   - 関連エージェントを呼び出す
   - 変更を実施
   - テスト

5. **プレビュー更新**
   - ホットリロードまたは再起動

---

## 使用例

```
/enhance ダークモードを追加
/enhance 管理パネルを構築
/enhance 決済システムを統合
/enhance 検索機能を追加
/enhance プロフィールページを編集
/enhance レスポンシブ対応
```

---

## 注意事項

- 大きな変更には承認を得る
- 競合するリクエストは警告する（例: PostgreSQL使用中に「Firebaseを使って」）
- 各変更をgitでコミット
