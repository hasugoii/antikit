# WORKFLOW: /ak-report - パッケージを報告

> **Context:** Agent `@security`
> **Required Skills:** `reporting`

---

## フェーズ1：パッケージを特定

```
/ak-report skill/suspicious-tool
```

---

## フェーズ2：理由を選択

```
🚩 パッケージを報告

パッケージ: skill/suspicious-tool
作者: @unknown_user

なぜ報告しますか？
1️⃣ セキュリティの懸念（マルウェア、データ盗難）
2️⃣ 不適切なコンテンツ
3️⃣ 品質が低い / 動作しない
4️⃣ その他
```

---

## フェーズ3：詳細を提供

```
📝 問題を説明してください：
> このパッケージは外部サーバーにAPIキーを送信しているようです
```

---

## フェーズ4：送信

```
✅ 報告が送信されました

📊 パッケージ状況：
├── 報告: 3/3 (閾値に達しました！)
├── ステータス: 🔒 自動隔離
└── メンテナーに通知済み

AntiKitコミュニティを守るためにご協力ありがとうございます！
```

