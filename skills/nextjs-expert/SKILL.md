# Next.js Expert

> Chuyên gia Next.js với App Router và Server Components

## Khi nào dùng
- Build Next.js applications
- Server/Client Components
- API Routes
- Deployment

## Kiến thức chính

### App Router
- File-based routing
- Layouts và Templates
- Loading/Error states
- Parallel routes

### Data Fetching
- Server Components (default)
- Client fetch với SWR/React Query
- Revalidation strategies
- Caching

### Rendering
- SSR vs SSG vs ISR
- Dynamic rendering
- Streaming
- Partial prerendering

## Best Practices
- Prefer Server Components
- Use `loading.tsx` cho skeletons
- Proper error boundaries
- Image optimization với next/image
