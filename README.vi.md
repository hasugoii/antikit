## 🤯 Nếu AI không chỉ code, mà còn bán hàng giùm bạn thì sao?

Phần lớn AI coding tool dừng ở việc viết code. **AntiKit thì không.**

```
Bạn: /brainstorm app quản lý chi tiêu
AI:  → Nghiên cứu thị trường + phân tích đối thủ
     → Chọn tính năng MVP + tech stack
     → Vẽ kiến trúc hệ thống

Bạn: /code
AI:  → Full-stack code có test
     → 21 agent chuyên gia cùng làm

Bạn: /launch
AI:  → Landing page (code xong + deploy luôn)
     → Bài đăng Facebook/TikTok/Zalo (copy-paste xài liền)
     → Thiết lập thanh toán (SePay, MoMo, VNPay)
     → Lịch marketing 30 ngày

Bạn: /grow
AI:  → Chiến lược giữ chân khách
     → Thiết kế chương trình giới thiệu
     → Tối ưu giá (phương pháp Van Westendorp)
     → Email sequences (copy-paste xài liền)
```

**Kết quả:** Từ ý tưởng đến doanh thu trong vài ngày, không phải vài tháng.

## 🎮 Slash Commands — Đội AI Của Bạn

Gõ lệnh. Triệu hồi chuyên gia. Đơn giản vậy thôi.

| Lệnh | Chuyện gì xảy ra | Bạn cảm thấy... |
|:------|:------------------|:-----------------|
| `/brainstorm` | 💡 Nghiên cứu thị trường, phân tích đối thủ, lên tính năng | "Giờ mình biết build gì rồi" |
| `/plan` | 📝 Kiến trúc, phân task, chọn tech stack | "Plan này chắc chắn lắm" |
| `/visualize` | 🎨 Thiết kế UI/UX có **tối ưu chuyển đổi** sẵn | "Nhìn pro thật" |
| `/code` | 💻 Full-stack code, multi-agent, có test | "Chạy luôn, khỏi sửa" |
| `/test` | ✅ Unit + E2E + A/B testing có thống kê | "Yên tâm ship" |
| `/deploy` | 🚀 Docker, CI/CD, analytics + SEO tự động | "Lên mạng rồi!" |
| `/audit` | 🏥 Scan bảo mật + SEO + performance | "An toàn và nhanh" |
| `/launch` | 📢 **Go-to-market: content, landing page, thanh toán** | "Có người đăng ký rồi!" |
| `/grow` | 📈 **Giữ chân, tối ưu giá, referral, email** | "Doanh thu tăng!" |

### 🔥 Pipeline `/launch` + `/grow` — *Lý do mọi người cài AntiKit*

```
/launch tạo cho bạn:
├── 📋 Lịch marketing 30 ngày
├── 📱 Bài Facebook/TikTok/Zalo (copy-paste xài liền)
├── 🏠 Landing page (code xong, 8 section chuyển đổi cao)
├── 💰 Thanh toán (SePay, MoMo, VNPay cho 🇻🇳)
├── 🚀 Kit launch Product Hunt (tagline, mô tả, maker comment)
└── 📧 Email sequences (chào mừng, onboarding, retention)

/grow tạo cho bạn:
├── 📊 Chấm điểm Product-Market Fit (Sean Ellis 40%)
├── 🔄 Chiến lược chống churn
├── 💎 Tối ưu giá (Van Westendorp, MaxDiff)
├── 🎁 Chương trình giới thiệu (thưởng 2 bên)
├── 📧 Email retention (copy-paste xài liền)
└── 📈 Báo cáo growth hàng tháng
```

## 🤖 21 Agent — Đội Ngũ AI Đa Chuyên Gia

Mọi task đều có **2+ agent** cùng làm. Không phải 1 AI — mà là **1 đội**.

```
Bạn: "Build trang login"

AntiKit:
🤖 CHÍNH: @frontend-specialist (React, CSS, accessibility)
🤖 HỖ TRỢ: @security-auditor (XSS, CSRF, JWT)
🤖 HỖ TRỢ: @test-engineer (E2E tests)

→ Kết quả: Trang login đẹp, an toàn, có test
```

| Bảo vệ | Làm gì |
|:--------|:-------|
| 🚫 **Chặn lệnh hủy diệt** | `rm -rf /` → **CHẶN** |
| 🔒 **Khóa trong dự án** | AI chỉ sửa file trong folder dự án |

| 🎯 **Chống lạc đề** | Tự phát hiện scope creep |
