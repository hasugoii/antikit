## 🎯 什么是 AntiKit？

**AntiKit**（Antigravity Kit）是 [Google Antigravity](https://ai.google.dev/aistudio) 的终极 **Vibe Coding** 工具。

> 您的个人AI开发操作系统 - workflows、agents和skills将您的想法转化为生产就绪的代码。

**适合希望：**
- 🚀 使用AI驱动的workflows更快交付
- 🧠 在会话之间永不丢失上下文
- 🛡️ 从第一天就写出生产就绪的代码

### 🧠 "AI全忘了！"

盯着屏幕不知道下一步该做什么。没有路线图，没有清晰的流程。只是随意聊天。

## ✅ 解决方案

**AntiKit通过规范化工作流解决所有这些问题。**

| 问题 | AntiKit解决方案 |
|------|-----------------|
| 🧠 AI遗忘 | `/save-brain` 将上下文保存到 `.brain/` |
| 💥 脆弱代码 | `/code` 包含安全检查和最佳实践 |
| 🔄 调试循环 | `/debug` 找到根本原因，而非修补 |
| ❓ 迷失方向 | `/next` 建议最佳下一步 |
| 🍝 混乱代码 | `/refactor` + `/audit` 保持代码整洁 |
| 😰 无备份 | `/rollback` + 自动Git集成 |

## 🚀 快速开始

### macOS / Linux

```bash
curl -fsSL https://raw.githubusercontent.com/hasugoii/antikit/main/install.sh | bash
```

### Windows (PowerShell)

```powershell
irm https://raw.githubusercontent.com/hasugoii/antikit/main/install.ps1 | iex
```

**安装后，尝试：**

```
/recap
```

## 📋 工作流程

AntiKit将开发组织成**4个清晰阶段**：

### 阶段1: 🎯 规划
> **先不要写代码！**

| 命令 | 用途 |
|------|------|
| `/init` | 设置项目结构、Git、环境 |
| `/recap` | 休息后恢复上下文 |
| `/brainstorm` | 规划前探索想法 |
| `/plan` | 设计数据库、API、创建规格 |
| `/next` | 卡住时获取建议 |

### 阶段2: 🔨 构建
> **安全且美观地编码**

| 命令 | 用途 |
|------|------|
| `/visualize` | 设计漂亮的UI布局 |
| `/code` | 带安全检查的代码编写 |
| `/run` | 运行应用程序 |

### 阶段3: ⚙️ 运维
> **维护与优化**

| 命令 | 用途 |
|------|------|
| `/debug` | 找到bug的根本原因 |
| `/test` | 运行测试 |
| `/refactor` | 清理代码 |
| `/rollback` | 回滚到之前版本 |
| `/deploy` | 部署到生产环境 |

### 阶段4: 🛡️ 管理
> **管理与改进**

| 命令 | 用途 |
|------|------|
| `/save-brain` | 保存项目知识 |
| `/config` | 配置skills和agents |
| `/audit` | 安全和代码审查 |
| `/ak-update` | 更新AntiKit |
