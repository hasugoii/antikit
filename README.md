## рҹ“ҡ AI That Learns From Mistakes

> *New in v1.14.0*

| Feature | What it does |
|:--------|:-------------|
| рҹ“қ **Lessons System** | AI saves what went wrong (and right) after every workflow |
| рҹ”Қ **Change Impact Awareness** | Before ANY change, AI must check ripple effects вҖ” **mandatory** |
| рҹҸҒ **Definition of Done Gate** | AI self-checks 6 quality points before reporting "done" |

```
/code  вҶ’ AI writes code вҶ’ DoD Gate: "Did tests pass? Is console clean? Would a senior approve?" вҶ’ вң… Done
/debug вҶ’ AI fixes bug   вҶ’ Lesson saved: "Don't modify X without updating Y"
/recap вҶ’ AI loads lessons from .brain/lessons.md вҶ’ Never makes same mistake twice
```

**The Change Impact Awareness rule prevents the #1 AI coding mistake:**
> *Creating a new file but forgetting to register it in router/config/index.*

Now AI asks **before every change:** *"What else depends on this?"*

## рҹ§  Permanent Memory (AI Never Forgets Again)

> *"I explained the whole architecture yesterday... and today it acts like we never talked."*

Sound familiar? **AntiKit solves this.**

| Problem | AntiKit Solution |
|:--------|:----------------|
| рҹ”Ӣ Token runs out mid-conversation | `/save-brain` preserves context вҖ” **never lose progress again** |

```

```
